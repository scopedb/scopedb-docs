# Query operators

ScopeDB supports many frequent operators, including arithmetic operators (such as `+` and `-`), logical operators (such as `AND` and `OR`), set operators (such as `UNION`), etc.

| Category                                         | Operators                       |
|--------------------------------------------------|---------------------------------|
| [Arithmetic operators](operators-arithmetic.mdx) | `+`, `-`, `*`, `/`, `%`         |
| [Comparison operators](operators-comparison.mdx) | `=`, `!=`, `>`, `<`, `>=`, `<=` |
| [Logical operators](operators-logical.mdx)       | `AND`, `OR`, `NOT`              |
| [Set operators](operators-set.mdx)               | `UNION`                         |
